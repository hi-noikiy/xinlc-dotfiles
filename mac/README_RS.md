
基于 Rust 编写的实用命令行工具
===

- 目标：统一主流操作系统的终端配置
- 策略：尽量使用 Rust 编写的工具链
- Rust 工具链优势
  - 配置文件为 TOML 或 YAML，声明式配置，可读性强
  - 多使用系统原生工具进行编译，需要下载安装的依赖少
  - 速度上较快，有时可能不如 C 工具链，但比其他语言的工具往往要快
  - 跨平台性优于 C

- https://lib.rs/command-line-utilities/
